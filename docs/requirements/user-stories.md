# User Stories – DFL App

---

# 1. Participant User Stories

## 1.1 Event & Account
- **P1:** As a participant, I want to see a list of upcoming DFL events, so that I can choose the right one for me.
- **P2:** As a participant, I want to register for an event directly in the app, so that my spot is reserved.
- **P3:** As a participant, I want to create an account, so that I can securely access my personal DFL content.
- **P4:** As a participant, I want to choose my preferred language, so that I can use the app comfortably.

## 1.2 Questionnaires (Gifts & Values)
- **P5:** As a participant, I want to fill out the spiritual gifts questionnaire, so that I can reflect on my giftedness.
- **P6:** As a participant, I want to fill out the values questionnaire, so that I can identify what matters most to me.
- **P7:** As a participant, I want my questionnaire progress to save automatically, so that I do not lose my work.
- **P8:** As a participant, I want to review a summary of my questionnaire results and define my Top 3, so that I understand my strengths clearly.

## 1.3 Life Tree & Reflection
- **P9:** As a participant, I want to use a specialized tree builder, so that I can easily map out my life journey (Roots & Branches).
- **P10:** As a participant, I want to identify my "3 red threads" in my Life Tree, so that I can see patterns in my history.

## 1.4 Lesson Notes & Takeaways
- **P11:** As a participant, I want to take notes and add photos of slides during teachings, so that I don't forget important points.
- **P12:** As a participant, I want to record 1–3 "Key Takeaways" for every module, so that I focus on the most important insights.

## 1.5 Listening Prayer
- **P13:** As a participant, I want to write down my listening‑prayer impressions, so that I can revisit what God may be saying.
- **P14:** As a participant, I want to see impressions others have received for me, so that I can be encouraged.
- **P15:** As a participant, I want to choose whether these impressions remain private or shared, so that I stay in control of my personal spiritual process.

## 1.6 Summary & Collage
- **P16:** As a participant, I want to create a visual collage of all my Key Takeaways, so that I see the "Big Picture" of my weekend.

## 1.7 Goals (SMART)
- **P17:** As a participant, I want to define 3 SMART goals, so that I can take concrete steps after the weekend.

## 1.8 Sharing & Privacy Controls
- **P18:** As a participant, I want to mark each item as *private* or *shared*, so that I control what leaders can see.
- **P19:** As a participant, I want to use a "Presentation Mode" to show my results on a screen without giving permanent access to my data.

# 2. Small Group Leader User Stories

## 2.1 Group Overview
- **L1:** As a leader, I want to see my group members and their completion status, so that I know who may need support. 

## 2.2 Access to Shared Participant Information
- **L2:** As a leader, I want to see only the information that participants explicitly shared, so that I respect their privacy.
- **L3:** As a leader, I want to view shared summaries of gifts, values, Life Tree, and listening prayer, so that I can guide discussions well.

## 2.3 1:1 Conversations & Feedback
- **L4:** As a leader, I want to take private notes about conversations, so that I can prepare well for 1:1 sessions.
- **L5:** As a leader, I want to explicitly share specific feedback or goal refinements with a participant, so that they can see it in their results.

---

# 3. Administrator / Event Manager User Stories

## 3.1 Event Setup
- **A1:** As an admin, I want to create a new event and define its location, rooms, and building plan, so that participants have all the info.
- **A2:** As an admin, I want to create a schedule using a DFL template, so that I don't have to start from scratch.
- **A3:** As an admin, I want to assign specific modules (e.g., Values, Gifts) to schedule slots, so that the app guides the participants.

## 3.2 User & Registration Management
- **A4:** As an admin, I want to manage registrations (approve/deny), so that I can control the participant list.
- **A5:** As an admin, I want to assign participants to leaders and groups, so that the structure matches the DFL setup.

## 3.3 Content & Materials
- **A6:** As an admin, I want to upload materials (PDF, images, texts), so that all content is in one place.

---

# 4. General System Stories
- **S1:** As a user, I want the app to work offline, so that I can use it even with poor retreat WiFi.
- **S2:** As a user, I want my data to sync automatically when I'm back online.
- **S3:** As a user, I want to be sure that my private notes are never visible to anyone else, including admins.
