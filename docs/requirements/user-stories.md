# User Stories – DFL App

---

# 1. Participant User Stories

## 1.1 Account & Profile
- **P1:** As a participant, I want to create an account, so that I can securely access my personal DFL content.
- **P2:** As a participant, I want to choose my preferred language, so that I can use the app comfortably.

## 1.2 Questionnaires (Gifts & Values)
- **P3:** As a participant, I want to fill out the spiritual gifts questionnaire, so that I can reflect on my giftedness.
- **P4:** As a participant, I want to fill out the values questionnaire, so that I can identify what matters most to me.
- **P5:** As a participant, I want my questionnaire progress to save automatically, so that I do not lose my work.
- **P6:** As a participant, I want to review a summary of my questionnaire results, so that I understand my strengths clearly.

## 1.3 Life Tree & Reflection
- **P7:** As a participant, I want to upload or create my Life Tree, so that I can reflect on my life journey.
- **P8:** As a participant, I want to add short notes to my Life Tree, so that I remember insights during discussions.

## 1.4 Listening Prayer
- **P9:** As a participant, I want to write down my listening‑prayer impressions, so that I can revisit what God may be saying.
- **P10:** As a participant, I want to choose whether these impressions remain private or shared, so that I stay in control of my personal spiritual process.

## 1.5 Notes & Personal Space
- **P11:** As a participant, I want a private notes space, so that I can capture thoughts that no one else can see.
- **P12:** As a participant, I want to keep all my insights (gifts, values, Life Tree, listening prayer) in one place, so that I see the big picture.

## 1.6 Schedule & Materials
- **P13:** As a participant, I want to view the DFL weekend schedule, so that I always know what is happening.
- **P14:** As a participant, I want to access teaching materials (PDFs, texts, images), so that I am well prepared.

## 1.7 Sharing & Privacy Controls
- **P15:** As a participant, I want to mark each item as *private* or *shared*, so that I control what leaders can see.
- **P16:** As a participant, I want to see a clear list titled “What my leader can see,” so that I stay aware of my privacy.
- **P17:** As a participant, I want to temporarily share an item (Presentation Mode), so that I can show it during the session without giving permanent access.
- **P18:** As a participant, I want temporary sharing to expire automatically, so that I do not accidentally leave sensitive content open.

## 1.8 Reference Feedback (Extended Feature)
- **P19:** As a participant, I want to invite a reference person to answer a simplified gifts questionnaire about me, so that I get external feedback.
- **P20:** As a participant, I want to choose whether the reference feedback is visible to my group leader, so that I maintain control.

# 2. Small Group Leader User Stories

## 2.1 Group Overview
- **L1:** As a leader, I want to see my group members, so that I know whom I am responsible for.
- **L2:** As a leader, I want to see whether participants have completed their questionnaires, so that I know who may need support. 


## 2.2 Access to Shared Participant Information
- **L3:** As a leader, I want to see only the information that participants explicitly shared, so that I respect their privacy.
- **L4:** As a leader, I want to view shared summaries of gifts, values, Life Tree, and listening prayer, so that I can guide discussions well.
- **L5:** As a leader, I want to see aggregated reference feedback (if shared), so that I understand the participant more holistically.

## 2.3 Personal Leader Notes
- **L6:** As a leader, I want to take notes about conversations, so that I can prepare well for 1:1 sessions.
- **L7:** As a leader, I want my notes to remain private, so that participants cannot see internal guidance or pastoral considerations.
- **L8:** As a leader, I want to write down my impression during my listening prayer for every Member and share this impression with this member

## 2.4 1:1 Session Planning
- **L8:** As a leader, I want to create and manage time slots for 1:1 conversations, so that the weekend runs smoothly.
- **L9:** As a leader, I want participants to sign up for available time slots, so that scheduling is easy.

## 2.5 Presentation Mode
- **L10:** As a leader, I want to see a participant’s content only when they consciously share it (temporary share), so that I never access anything unintentionally.

# 3. Administrator User Stories

## 3.1 User & Group Management
- **A1:** As an admin, I want to create and manage user accounts, so that the right people have access.
- **A2:** As an admin, I want to assign participants to leaders and groups, so that the structure matches the DFL setup.

## 3.2 Content & Materials
- **A3:** As an admin, I want to upload materials (PDF, images, texts), so that all content is in one place.
- **A4:** As an admin, I want to update materials at any time, so that changes are reflected instantly.

## 3.3 Translations & Event Settings
- **A5:** As an admin, I want to manage translations/languages, so that DFL works in multilingual environments.
- **A6:** As an admin, I want to edit the weekend schedule, so that updates display immediately in the app.

## 3.4 Privacy Management
- **A7:** As an admin, I want to configure visibility defaults (e.g., “private by default”), so that the app respects spiritual and personal sensitivity.
- **A8:** As an admin, I want audit logs of leader access, so that privacy compliance is ensured.

---

# 4. General System Stories

## 4.1 Security & Access
- **S1:** As a user, I want secure login with password reset, so that my data is protected.
- **S2:** As a user, I want my data encrypted, so that no one unauthorized can access it.

## 4.2 Offline & Sync
- **S3:** As a user, I want questionnaires and notes to work offline, so that weak retreat WiFi is not a barrier.
- **S4:** As a user, I want my offline work to sync automatically when online, so that everything is up to date.

## 4.3 Notifications
- **S5:** As a user, I want optional reminders (e.g., “Complete your values test before Session 3”), so that I stay on track.
