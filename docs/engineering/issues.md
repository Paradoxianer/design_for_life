## ⚡ ✨ #16: Implement Life Tree Editor [prio: 2, feature] 🏁 [Release 1 (MVP)]
---
**Status / Description:**

### 🌳 Life Tree Editor: Detailed Requirements

**1. Hybrid Entry Mode (Notes Extension)**
- Start screen is inherited from the Notes Editor  (Text field + Image upload (for analog drawings)).
- A specific 'Life Tree' icon integrated next to image actions to launch the Visual Editor.
- Standard 'Top 3 Key Takeaways' fields included at the bottom of the editor since its derived from Notes Editor.


**2. Intuitive In-Place Graph Editing**
- **No Dialogs**: Tapping a node transforms it into an input field directly within the graph view.
- **Editable Root**: Starts with 'Geburt' (Birth), which can be renamed/extended. 

**3. Smart Expansion (Ghost Nodes)**
- When a node is active/edited, 'Ghost Nodes' (placeholders) appear automatically:
    - 'Virtual Child' below the node.
    - 'Virtual Sibling' to the side.
- Tapping a ghost node instantly creates a real node and shifts focus to its text field.
- The first node is Birth and should be treaded specially :) (like if there is one named children when edited it should add a second "gost Node" as children) 

**4. Unified Result View**
- Polished read-only view displaying analog content (image/text) and the digital tree structure.

**5. Share Posibilty**
- implements the share Resulst posibiility user can choose to share.. the text, the picture or a picture of the digital tree structure
