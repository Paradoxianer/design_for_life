# Release Notes – DFL App

## [v1.0.0-MVP] – Preparation
**Status:** In Development

### Added
- **Event Discovery:** Browse and register for DFL weekends.
- **Unified Modules:** Questionnaire engine for Gifts and Values assessments.
- **Life Tree:** Interactive tree builder for life journey mapping.
- **Lesson Notes:** Integrated note-taking for teachings with photo support.
- **Listening Prayer:** Digital capture of spiritual impressions.
- **Collage Mode:** Visual "Big Picture" dashboard based on Key Takeaways.
- **SMART Goals:** Concrete goal-setting tool.
- **Privacy Core:** 4-Step sharing dialog and Presentation Mode.
- **Admin Tools:** Event creation, room management, and schedule editing.

### Technical Improvements
- **Offline First:** Local SQLite encryption and background sync.
- **Multilingual:** Full support for German and English.
- **Cross-Platform:** Deployment targets for iOS, Android, and Web.
- **Security:** Field-level encryption for Prayer Notes.

- **Group Photos:** Automated generation of event memories.
