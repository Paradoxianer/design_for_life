# Admin & Organizer Guide – DFL App

## 1. Einleitung
Dieses Dokument unterstützt Administratoren und Organisatoren bei der technischen Vorbereitung eines Design for Life (DFL) Wochenendes. Fokus: Maximale Einfachheit.

## 2. Event-Management

### 2.1 Ein neues Event anlegen
1. Öffnen Sie die App und gehen Sie zum **Event Selection Screen**.
2. Tippen Sie auf das **"+" Symbol** in der App-Bar.
3. Geben Sie Titel, Beschreibung, Ort und Datum ein.
4. **Raumplanung (Simple):** Es gibt keinen separaten Screen für Räume. Geben Sie den Raum einfach direkt bei der Session im Zeitplan ein (Textfeld mit Autovervollständigung).

### 2.2 Zeitplan (Schedule) bearbeiten
1. Wenn Sie ein neues Event erstellen, ist der **Standard-DFL-Ablauf bereits voreingestellt geladen**.
2. Sie müssen nur die Uhrzeiten anpassen oder bei Bedarf Sessions verschieben.
3. Jede Session hat ein Feld "Raum", in das Sie den Ort (z.B. "Plenum" oder "Raum 102") eintragen können.

## 3. Teilnehmer- & Gruppenverwaltung

### 3.1 Registrierungen & Rollen
1. In der Teilnehmerliste können Sie pro Person die Rolle (Teilnehmer, Leiter, Admin) festlegen.

### 3.2 Kleingruppen zuweisen (Simple Dropdown)
1. Es gibt keinen separaten Gruppen-Screen.
2. In der Teilnehmerliste befindet sich hinter jedem Namen ein **Dropdown-Feld** für die Gruppe.
3. Wählen Sie eine bestehende Gruppe (z.B. "1" oder "2") aus.
4. Möchten Sie eine neue Gruppe erstellen, tippen Sie einfach auf das **"+" am Ende der Dropdown-Liste**.

## 4. Materialien (Nur für Leiter)
- **Sichtbarkeit:** PDFs, Bilder und Vorlagen sind **nur für Gruppenleiter** bestimmt (zur Vorbereitung). Teilnehmer sehen diese Inhalte nicht in der App.
- **Vorlagen:** Bei der Erstellung eines neuen DFL-Wochenendes können Sie Materialien früherer Kurse einfach als Vorlage übernehmen.
- **Zukunft:** Materialien sollen später direkt in der Material-Ansicht bearbeitet werden können.

## 5. Support während des Events
- **Sync-Status:** Prüfen Sie bei Problemen das Sync-Icon (unten rechts).
- **Magic Link:** Falls der Login fehlschlägt, kann der Admin in der Userverwaltung einen neuen Magic Link anfordern.
